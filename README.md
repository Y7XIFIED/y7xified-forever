## 🚀 Live Website
Experience the fully interactive website live here: https://y7xified-forever.vercel.app

## 🎮 How to Control
When the page loads, simply press the **Spacebar** to unleash the ball and start breaking the blocks! Watch as it bounces around, changing the colors of the blocks in an engaging interactive display.

## ⚙️ Installation

**Prerequisites:**
- Node.js (v18+)
- pnpm

**Setup Instructions:**
1. Clone the repository: `git clone https://github.com/Y7XIFIED/y7xified-forever.git`
2. Navigate into the folder: `cd y7xified-forever`
3. Install dependencies: `pnpm install`
4. Start the dev server: `pnpm run dev`
